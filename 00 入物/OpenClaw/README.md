# OpenClaw 自动化笔记

这是由 OpenClaw 助手自动生成和管理的内容文件夹。

## 📁 文件夹结构

| 文件夹 | 用途 |
|--------|------|
| `技术新闻/` | 自动抓取的 Linux/AI 技术新闻 |
| `AI 研究/` | AI 论文、技术文章摘要、可视化图表 |
| `Arch Linux/` | Arch 配置、技巧、问题解决方案 |
| `临时收集/` | 未分类的临时内容，待整理 |

## 🤖 自动化能力

- **每日技术新闻推送** - 9:00 AM 自动抓取并创建笔记 + 可视化图表
- **AI 动态追踪** - 自动总结最新 AI 研究
- **学习笔记归档** - 对话中提到的重要内容自动保存
- **可视化生成** - Mermaid/Canvas/Excalidraw 图表自动生成

## 📝 使用说明

1. 所有内容会自动分类到对应子文件夹
2. 文件名格式：`YYYY-MM-DD-标题.md`
3. 定期查看 `临时收集/` 并手动整理到 vault 其他位置
