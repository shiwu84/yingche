# Network MOC

> [!summary] 网络配置知识地图
> 本文介绍如何在 Arch Linux 上配置 OSI 第 3 层及以上的网络连接，包括 IP 地址、路由、DNS 和网络管理。

## 核心概念

- [[Network-网络接口]] - 网络接口类型、命名规则、管理方法
- [[Network-IP 地址配置]] - 静态 IP、DHCP、iproute2 命令
- [[Network-路由表]] - 路由配置、默认网关、路由表管理
- [[Network-DNS 配置]] - 域名解析、/etc/resolv.conf、DNS 服务器
- [[Network-网络管理器]] - NetworkManager、systemd-networkd、netctl 等
- [[Network-无线网络]] - WiFi 配置、wpa_supplicant、iwd
- [[Network-网络诊断]] - ping、traceroute、故障排除工具
- [[Network-网络启动]] - 开机自动连接、持久化配置
