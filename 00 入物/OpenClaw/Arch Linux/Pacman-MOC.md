# Pacman MOC

> [!summary] 包管理器知识地图
> Pacman 是 Arch Linux 的包管理器，用于安装、更新、移除和管理软件包。它结合了简单的二进制包格式和易用的 Arch 构建系统。

## 核心概念

- [[Pacman-安装软件包]] - 安装单个或多个包，包括依赖处理
- [[Pacman-移除软件包]] - 移除包和不再需要的依赖
- [[Pacman-升级系统]] - 完整系统升级（-Syu）
- [[Pacman-查询数据库]] - 搜索和查询包信息
- [[Pacman-清理缓存]] - 管理包缓存，释放磁盘空间
- [[Pacman-配置文件]] - /etc/pacman.conf 配置选项
- [[Pacman-故障排除]] - 常见问题和解决方案
